# City Scan Data Wiki

A reference guide to the data layers used in the [City Scan](https://github.com/cityresilience/city-scan) open-source geospatial tool.

## Overview

The City Scan Data Wiki documents every data layer included in City Scan — what it is, where it comes from, how it's made, and how to interpret it.

## About City Scan

City Scan is a rapid geospatial assessment tool that uses publicly available global datasets to produce maps, data, visualizations, and insights about the built and natural environments of a city or area of interest.
